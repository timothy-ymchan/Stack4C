#STACK4C
Stack4C is a simple Stack library written by a beginner, which can easily create stack of int, char and float
Descriptions are written in the header, read it yourself

Using it in Code::Blocks:
1) Open a console project and throw the files next to the main.c
2) In build option>search directories>Add, add the current path as search directories (In Relative path)
3) Add the Stack.c file to the project
4) Have fun!
